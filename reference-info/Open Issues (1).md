## List of Tags

The following are actively used at the moment:

Most of them should be self-explanatory, and for the others, here are the explanations:

- *bounty*
	There is a *bounty* put on these issues! Typically, these are handled via [FOSS Factory](https://www.gnu.org/software/hurd/donate.html#FOSS_Factory).
- *fixed\_in\_debian*
	This tag is used to tag items that have been fixed in the [Debian GNU/Hurd](https://www.gnu.org/software/hurd/hurd/running/debian.html) distribution, but not yet in the upstream sources.
- *open\_issue\_documentation*
	Used for tagging pages / items that need to be handled / improved for documentation purposes.
- *open\_issue\_porting*
	A list of open issues in porting software to run on GNU/Hurd systems. This list also includes [toolchain](https://www.gnu.org/software/hurd/toolchain.html) -level items, items that are either already solved in [Debian GNU/Hurd](https://www.gnu.org/software/hurd/hurd/running/debian.html) systems (tagged *fixed\_in\_debian*) or being worked around, so if you're out for working on application-level porting issues, then perusing through the list of [Debian packages that need porting](https://www.gnu.org/software/hurd/hurd/running/debian/porting.html) may be better.
- *stable\_URL*
	These pages are tagged as having a *stable URL*. That is, they're linked to from external pages, and their locations should not be changed needlessly.
